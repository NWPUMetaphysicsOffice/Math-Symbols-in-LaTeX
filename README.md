## Usage
Add `\input{math-symbols}` in your tex document.

## Note
* 整理了一些简单的数学宏定义方便写作, 更多问题请参见[readme.pdf](readme.pdf)
* 命名规则基于"所读即所写"(WYRIWYT, abbr of _W_hat _Y_ou _R_ead _I_s _W_hat _Y_ou _T_ype)
